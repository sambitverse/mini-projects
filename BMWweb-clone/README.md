# BMW M Performance Web Clone

A premium, sleek, and high-performance landing page clone inspired by the athletic and iconic design of the BMW M division (M Performance). 

## 🏎️ Features
- **Cinematic Hero Section:** An immersive, full-screen background cover with a subtle overlay, bold typography, and dual call-to-action buttons.
- **Featured Vehicles Gallery:** Responsive grid layout showcasing premium high-performance models (Coupe X, Sport GT, and Performance SUV) with hover translation animations.
- **Performance Engineering:** A detailed showcase of advanced engineering features with clean list items.
- **Motorsport Banner:** A bold call-to-action section built for pure track-day enthusiasts.
- **Ultimate Driving Experience:** A modern grid of custom experience cards (Test Drive, Track Days, Customization, Ownership) featuring hover slide animations.
- **Fully Responsive Layout:** Optimized navigation and structure supporting desktop, tablet, and mobile screens.

## 🛠️ Tech Stack
- **HTML5:** Semantic architecture (`<header>`, `<nav>`, `<section>`, `<footer>`).
- **CSS3:** Native CSS Grid & Flexbox, smooth scrolling behavior, and custom transition micro-animations.
- **Typography:** Sleek sans-serif presentation.
- **Theme:** Ultra-premium dark mode aesthetic (`#111` background, vibrant `#0077ff` BMW accent blue, and pure white).

## 🚀 How to Run Locally
1. Clone the repository (if you haven't already):
   ```bash
   git clone https://github.com/sambitverse/mini-projects.git
   ```
2. Navigate to the project directory:
   ```bash
   cd mini-projects/BMWweb-clone
   ```
3. Open `index.html` in your favorite web browser.
